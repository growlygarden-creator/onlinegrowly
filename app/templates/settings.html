Koblingsskjema

Dette er oppsettet som faktisk har fungert i prosjektet nå. Bruk det som fasit når du flytter alt over i en egen boks.

ESP32 → MAX485

3V3    → VCC
GPIO17 → DI
GPIO18 → RO
GPIO16 → DE
GPIO16 → RE
GND    → GND

DE og RE skal ligge på samme GPIO. Felles jord er helt nødvendig.

MAX485 → 7in1 jordsensor

A      → gul/grønn
B      → blå
Sensor brun  → +5V
Sensor svart → GND

Hvis jordsensoren gir timeout, er første test alltid å bytte A og B.

I2C-buss

GPIO8 → SDA
GPIO9 → SCL
3V3   → VCC
GND   → GND

BH1750 og BMP280 deler samme SDA og SCL på denne bussen.

BH1750 lys

VCC → 3V3
GND → GND
SDA → GPIO8
SCL → GPIO9
ADDR → åpen

BH1750 blir funnet på 0x23 i dette oppsettet.

BMP280 luftsensor

VCC → 3V3
GND → GND
SDA → GPIO8
SCL → GPIO9
CSB → 3V3
SDO → GND

Modulen du har nå svarer som BMP280 på 0x76. Den gir temperatur og lufttrykk, men ikke luftfuktighet.
